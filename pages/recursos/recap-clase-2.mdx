# Recap · Clase 2 · EXPERIMENTAR

Para releer en 5 minutos después de la clase.

## Lo esencial

1. **Cinco patrones cubren casi todo:** extraer (un archivo), sintetizar (una carpeta), plantilla (formato fijo), imagen (Ctrl+V), web (investigar).
2. **Los patrones se combinan** en un solo pedido. Ahí está la potencia real.
3. **Agentes = paralelo.** Muchas tareas parecidas e independientes, todas a la vez.
4. **La síntesis no se negocia:** todo trabajo con agentes termina en "sintetizá los hallazgos en...".
5. **Decile siempre dónde guardar** el resultado.

## Los cinco patrones, en cinco prompts

```
Leé @notas.md y extraé las tareas pendientes en un archivo nuevo
```
```
Mirá @comentarios/ y sintetizá los temas que se repiten
```
```
Armá el informe con @sintesis.md siguiendo @plantillas/informe.md
```
```
[Ctrl+V con una imagen] ¿Por qué esto no funciona? Sé brutal.
```
```
Buscá en la web [tema] y traeme 5 aprendizajes concretos con fuentes
```

## El prompt de agentes de la clase

```
Tres agentes en paralelo: uno analiza los comentarios de lectores,
otro los datos del INCAA, otro las newsletters viejas. Cuando terminen,
sintetizá todo en ordenado/diagnostico-final.md
```

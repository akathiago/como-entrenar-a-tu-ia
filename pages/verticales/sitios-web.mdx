## Vertical 1 · Sitios web y plataformas personales

**Para quién:** cualquiera que necesite existir online con identidad propia: freelancers, profesionales independientes, proyectos culturales, emprendimientos, docentes con cátedra.

## Por qué Claude Code y no una plantilla

Wix, Squarespace y compañía te alquilan una plantilla que usan otras 80.000 personas. Con Claude Code el sitio es tuyo: el diseño sale de una conversación sobre quién sos, los archivos te pertenecen y no pagás suscripción por tenerlo online (Vercel es gratis para proyectos personales).

## Caso 1: el portfolio que te da vergüenza no tener

```
Quiero mi sitio personal. Soy [tu profesión]. Entrevistame para
definir qué secciones necesita, qué tono tiene que tener y qué
trabajos míos mostrar. Después armá REQUISITOS.md y esperá mi ok.
```

El truco está en la entrevista: no le pidas "un portfolio", dejá que te saque la información. El resultado tiene tu voz, no la de la plantilla.

## Caso 2: el sitio de un proyecto o emprendimiento

```
Armá una landing de una página para [tu proyecto]: qué es, para
quién, cómo contactar. Sobria, rápida, que se lea perfecta en el
celular. Generame 3 previews de estilos distintos antes de construir.
```

## Caso 3: una publicación editorial

Para newsletters con archivo público, revistas digitales chicas, blogs con dignidad estética:

```
Quiero una publicación digital para mis textos. Cada texto es un
archivo markdown en una carpeta. El sitio los muestra como una
revista: portada con los últimos, página por texto, tipografía
editorial seria. Armá la estructura y cargá estos tres textos de
ejemplo: @textos/
```

Escribís en markdown, Claude publica. Tu flujo de escritura no cambia.

## Mini-proyecto para hoy

Una página `quien-soy.html`: nombre, qué hacés, tres trabajos, contacto. En blanco y negro. De la primera versión a algo que te guste: cuatro o cinco iteraciones. Cuando te guste, seguí el loop de la [Clase 3](/clase-3) y publicalo.

## Límites honestos

- **Tiendas online con pagos**: se puede, pero el manejo de pagos y stock te mete en terreno técnico de verdad. Para vender poco, un link de Mercado Pago dentro de tu sitio resuelve.
- **Sitios con usuarios y login**: posible, pero ya no es un proyecto de tarde. Andá con el loop de la Clase 3 y paciencia.
- **El dominio propio** (tudominio.com.ar) se compra aparte (~USD 10 por año) y se conecta a Vercel. Claude te guía.

# Las 5 verticales

Todo lo que se puede hacer con Claude Code sin saber programar entra, con buena voluntad, en cinco familias. Estas guías son el mapa: para quién es cada una, casos concretos con prompts listos para robar, y un mini-proyecto para arrancar hoy.

| Vertical | En una frase | Empezá por acá si... |
|----------|--------------|----------------------|
| [Sitios web y plataformas](/verticales/sitios-web) | Existir online sin pelearte con WordPress | Necesitás portfolio, sitio de proyecto o publicación propia |
| [Datos y dashboards](/verticales/datos-y-dashboards) | Del CSV caótico al gráfico que cuenta la historia | Trabajás con planillas, encuestas o datos públicos |
| [Automatización](/verticales/automatizacion) | Lo que hacés todos los lunes, que se haga solo | Tenés tareas repetitivas que odiás |
| [Agentes sobre corpus propio](/verticales/agentes-propios) | Una IA que conoce TUS documentos | Acumulaste años de PDFs, fallos, papers o apuntes |
| [Herramientas y juegos](/verticales/herramientas-y-juegos) | Quizzes, trivias y material didáctico jugable | Enseñás, capacitás o divulgás |

Las verticales se cruzan todo el tiempo: un dashboard vive en un sitio, un agente alimenta una automatización. No elijas una para siempre: elegí por dónde entrar.

## Qué es Claude Code

- **Tiempo de lectura:** 5 minutos

## La explicación corta

Claude Code es una IA que trabaja **en tu computadora**, no en una pestaña del navegador.

- **No es solo para programar.** Es un asistente de propósito general: lee archivos, analiza datos, escribe documentos, arma sitios web.
- **Vive en la terminal**, pero vos solo tenés conversaciones. La terminal es la ventanita negra, nada más que eso.
- **El único comando que tenés que saber de memoria:** `claude`

## "Pará, ¿la terminal? Eso es de hackers"

No. La terminal es una forma de hablarle a tu computadora escribiendo en vez de haciendo clic. Durante décadas fue territorio de programadores porque había que saberse los comandos. Eso se terminó: ahora escribís en castellano lo que querés y Claude se encarga del resto.

El cambio de cabeza es este:

> De "tengo que aprender comandos" a "tengo que saber qué quiero".

No estás aprendiendo a programar. Estás aprendiendo a **dirigir**: describís el resultado que necesitás y la máquina resuelve el cómo.

## Claude Code vs el chatbot de siempre

Ya usaste ChatGPT o la app de Claude. Esto es otra cosa:

| | Chatbot (ChatGPT, claude.ai) | Claude Code |
|---|---|---|
| **Acceso a tus archivos** | Le subís de a uno, con paciencia | Lee y escribe carpetas enteras |
| **Memoria** | Se olvida entre sesiones | Persistente, con un archivo de memoria por proyecto |
| **Ejecución** | Te dice qué hacer | Lo hace |
| **Muchos archivos** | De a uno | Decenas a la vez |
| **Trabajo en paralelo** | Una conversación | Varios agentes trabajando al mismo tiempo |

**La diferencia de fondo:** el chatbot te da consejos, Claude Code hace el trabajo con vos. Es la diferencia entre pedirle indicaciones a alguien y que se suba al auto a manejar.

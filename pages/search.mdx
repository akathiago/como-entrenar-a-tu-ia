# Buscar

Usá el buscador para encontrar cualquier cosa en el material del taller. Funciona directo en tu navegador, sin mandar nada a ningún servidor.

<link rel="stylesheet" href="/pagefind/pagefind-ui.css" />
<div id="search"></div>
<script src="/pagefind/pagefind-ui.js"></script>
<script dangerouslySetInnerHTML={{__html: `
  if (typeof window !== "undefined") {
    window.addEventListener("DOMContentLoaded", () => {
      new PagefindUI({ 
        element: "#search", 
        showImages: false, 
        baseUrl: "/",
        showSubResults: true
      })
    })
  }
`}} />

## Tips de búsqueda

- **Frases exactas**: usá comillas, como `"memoria del proyecto"`
- **Varias palabras**: buscá `agentes paralelo` para encontrar páginas con ambas
- **Secciones**: cada resultado te muestra de qué parte del material viene

## ¿No encontrás algo?

- Probá con sinónimos o con el término en inglés (prompt, deploy, etc.)
- Recorré la navegación de la barra lateral
- Empezá por [Antes de empezar](/antes-de-empezar/que-es-claude-code) si recién llegás

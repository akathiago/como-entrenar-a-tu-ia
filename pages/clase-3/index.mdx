## Dónde estamos en la historia

En la Clase 2 encontraste la pista: entre las notas del editor anterior había una idea enterrada. *"¿Y si armamos algo tipo test: qué clásico del cine argentino sos? A la gente le encantan esas cosas. Nunca tuve tiempo de hacerlo."*

Los datos te dicen que los lectores encuentran a Butaca "correcta pero olvidable". Una herramienta interactiva es exactamente el tipo de cosa que no se olvida. En esta clase la construís y la publicás.

**¿Preferís hacer otra cosa?** Perfecto: el proceso es idéntico para cualquier proyecto de las 5 verticales. El quiz es el ejemplo guiado; tu proyecto propio es el objetivo real.
